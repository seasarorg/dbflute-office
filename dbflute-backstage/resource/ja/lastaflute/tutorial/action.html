Action Convention

Naming and URL

Actionクラス
[Packageキーワード] + [Actionキーワード] + Action
URLの構成
Actionクラス + Executeメソッド

Action Eaxmple

profile.ProfilePasswordAction#change() ※クラス名 = Profile + Password + Action
/profile/password/change/
product.ProductAction#list() ※Actionキーワードを省略: Packageキーワードのみのクラス名に
/product/list/
product.ProductListAction#index() ※Executeメソッド名を省略: index()メソッドが省略の予約語
/product/list/
mypage.MypageAction#index() ※両方とも省略: 機能ごとのエントリページによく利用される
/mypage/
MypageAction#index() ※Packageキーワードを省略: Web Root の Package 直下に配置
/mypage/
RootAction#index() ※URL '/' を示す特別な Action
/

make Action Class

Action Defintion

Super Class
extends [App]BaseAction
Behavior DI
instance variable with @Resource
Execute Method
public method with @Execute
e.g. Action Class Definition @Java
public class ProductListAction extends DocksideBaseAction { // Super Class

    @Resource
    private ProductBhv productBhv; // Behavior DI

    @Execute
    public HtmlResponse index() { // Execute Method
        ...
    }
}

URL Parameter

e.g. accept URL parameter as arguments with urlPattern @Java
    @Execute(urlPattern = "{pageNumber}")
    public HtmlResponse index(Integer pageNumber) { // /product/list/3/
        ...
    }

e.g. accept URL parameter with more URL @Java
     // /product/list/oneman/ikspiary/mystic/
    @Execute(urlPattern = "{land}/ikspiary/{sea}")
    public HtmlResponse index(String land, String sea) {
        ... // land=oneman, sea=mystic
    }

Action Form (for POST, GET parameter)

e.g. accept POST parameter by action form @Java
    @Execute
    public HtmlResponse doSignin(SinginForm form) { // submit
        ...
    }

e.g. both URL parameter and POST parameter @Java
    // e.g. /.../both/3/?favoriteCode=sea&nextName=land
    @Execute(urlPattern = "{pageNumber}")
    public HtmlResponse both(Integer pageNumber, SinginForm form) {
        ...
    }

Execute Return

HtmlResponse
asHtml(path_...) *with FreeGen
JsonResponse
asJson(bean)
StreamResponse
asStream(ins)
e.g. HtmlResponse @Java
    @Execute
    public HtmlResponse index() {
        ...
        return asHtml(path_MyPage_MyPageJsp);
    }

e.g. JsonResponse @Java
    @Execute
    public JsonResponse product() {
        ProductBean bean = ...
        return asJson(bean);
    }

TemplatePath FreeGen (if use)

まず、Templateファイル (e.g. JSP) を作成 (この時点では中身は空っぽでOK)

配置場所
対応する Action と同じディレクトリ構成で e.g. WEB-INF/view/profile/
ファイル名
Formタグが飛ばすURLに対応 e.g. profile_password_change.jsp

そして、DBFlute の FreeGen を叩く！ (パス定義が自動生成される) 

e.g. Template File Location @Directory

    |  |
    |  |-mypage
    |  |  |-mypage.jsp // /mypage/
    |  |
    |  |-root.jsp // '/'
    |
    |-web.xml

e.g. you can use Template Path for HtmlResponse @Java
    @Execute

        ...
