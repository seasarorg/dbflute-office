DBFlute Exampleとは？

DBFluteの環境周りおよび実装周りの理解を一層深めてもらうための、実装にDBFluteを利用したEclipseプロジェクトです。Subversion からチェックアウトすることが可能で、実際にDBFluteのタスクの実行や、Example実装(JUnitのテストケース)を実行することができます。

(ほとんどの)Exampleは、スキーマ構造として ExampleDB を利用しています。

ExampleDB

Gitでお手軽ワンセット (まずはこちら)

DBFluteを知る上で必要な最小限のセットの Example、JUnitですぐさまお試しプロジェクト、WEBアプリと合わせたプロジェクトなどを用意しています。 (SVNの方にはないExampleもあり)

Git提供のExampleプロジェクト

dbflute-quick-stage
  • JUnitで気軽に実行できる Example
  • チェックアウトするだけで、ビルドもデータベース(H2)も準備OK
  • Seasar(S2Container) を利用している
  • 詳しくは、README_ja.txt を参照
maihama-base, common, dockside, saflute
  • WEBアプリと合わせた Example
  • M2E (Maven の Ecipseプラグイン) 使ってビルドする
  • データベースは MySQL を利用、3306で用意されていれば ReplaceSchema でDB構築できる
  • Docksideプロジェクトで Run Jetty Run で実行すれば画面起動できる
  • SAStruts に加え SAFlute を利用している
  • DBFluteの FreeGen の機能をふんだんに利用している
  • 詳しくは、README_ja.txt を参照
SAFlute
dbflute-hands-on
  • DBFluteハンズオンのためのプロジェクト

SVNで目的別 (ピンポイントで選ぶ)

Exampleどころか、DBFlute本体のソースも置いてあるリポジトリです。多種多様な目的に合わせた多くの Example が用意されていますので、一気にチェックアウトすると大変なので選びましょう。
