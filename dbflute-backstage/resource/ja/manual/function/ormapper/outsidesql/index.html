外だしSQLの使い方
外だしSQLの作成方法や手順など ※わからなくなったらとりあえずここ
  1. 外だしSQLを パラメータコメント を使って 2Way-SQL で書く
  2. Sql2Entity を実行して CustomizeEntity と ParameterBean を自動生成
  3. Behaviorの outsideSql() メソッドから実行
