DFEditorとは？

DBFluteプロパティ(dfprop)のファイルの設定支援を行う EMecha の中の一機能(EMechaがインストールされていれば利用可能)。これを利用すると、DBFluteプロパティが親密な友達になります。

DBFluteプロパティ

dfpropへの関連付け

dfpropへの関連付けは、インストールした時点で自動で設定されます。

色付け

DBFluteプロパティにおけるキーワードなど色付けがされます。

DBFluteプロパティを修正する人だけでなく、閲覧する可能性のある人にも利用価値があります。

エラーチェック

括弧の対応や、mapにおける重複キーのチェックなど、dfpropの構文チェックがされます。非常に大事な機能です。 この機能が有効なバージョンを利用することが推奨されます。@since EMecha-0.5.6

アウトライン表示

アウトライン(Outline)ビューにて、dfpropの構造が表示されます。特に、classificationDefinitionMap.dfprop や littleAdjustmentMap.dfprop においては、設定を鳥瞰するのにとても有用です。ぜひ Outline ビューを表示しましょう。@since EMecha-0.5.6
