IsNull

${indexlist}

概要

基本概念

null かどうかの条件(is null)を設定します。IsNull は、絞り込み条件を表す ConditionKey です。

ConditionBeanについて - ConditionKeyとは？

会話上では、いずぬる と表現します。

実装方法

実装の流れ

query() の後、set[column-name]_IsNull() を呼び出し、条件値を指定します。 

e.g. IsNull条件の実装手順 (Eclipseでコード補完) {BIRTHDATE} @Java
MemberCB cb = new MemberCB();
cb.q // .q と打って enter
--
cb.query()
--
// 1. .set まで打つとカラム選択
// 2. B (Birthdate) でカラム確定
// 3. INu (IsNull) で enter
cb.query().setBINu
--
cb.query().setBirthdate_IsNull();

メソッド仕様

基本仕様

条件値が存在しません。同じカラムに対して複数の設定処理をしても単なる上書きになります。

サポートされる型・カラム

全ての型に対してサポートされます。 ただし、PK以外のNotNull制約のカラムにはメソッドは生成されません。 (PKに関しては、関連テーブルの存在判定などで利用する可能性があるため生成されます)
