IsNotNull

${indexlist}

概要

基本概念

null でないかどうかの条件(is not null)を設定します。IsNotNull は、絞り込み条件を表す ConditionKey です。

ConditionBeanについて - ConditionKeyとは？

会話上では、いずのっとぬる と表現します。

実装方法

実装の流れ

query() の後、set[column-name]_IsNotNull() を呼び出し、条件値を指定します。 

e.g. IsNotNull条件の実装手順 (Eclipseでコード補完) {BIRTHDATE} @Java
MemberCB cb = new MemberCB();
cb.q // .q と打って enter
--
cb.query()
--
// 1. .set まで打つとカラム選択
// 2. B (Birthdate) でカラム確定
// 3. INN (IsNotNull) で enter
cb.query().setBINN
--
cb.query().setBirthdate_IsNotNull();
