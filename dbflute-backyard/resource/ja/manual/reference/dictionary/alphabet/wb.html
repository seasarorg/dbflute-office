Base

DBFluteにおいては、ジェネレーションギャップのスーパークラスを示す言葉として利用されています。 その他一般的な用語としても利用しています。

GenerationGap (ジェネレーションギャップ)

BasicValueType (基本ValueType)

型で管理される基本的なValueType。マッピングされているプログラム型に関連付いていて、SQL実行時に プラグインValueType の指定がなければ、該当プロパティのプログラム型に関連付く ValueType を利用してバインド変数処理やデータ取得処理を行います。

DBFluteConfig - ValueType
ValueType
PluginValueType (プラグインValueType)

Bs

ジェネレーションギャップのBsクラスを示す言葉 Base の略称。DBFluteでは、基本的にBsクラスのクラス名は BsXxx という形式になっています。また、それぞれのEntityのBsクラスをドキュメント上などで抽象的に示すために BsEntity というように利用されます(その他、BsBehavior、BsConditionBeanなど)。

BsClass (Bsクラス)
GenerationGap (ジェネレーションギャップ)

BsClass (Bsクラス)

DBFluteでジェネレーションギャップパターンで生成される Entity, Behavior, ConditionBean などのスーパークラスを示します。これらクラスはテーブル構造との同期を第一にしているため、再自動生成時には常に上書きされ、 アプリケーション独自のメソッドを追加することはできません。
