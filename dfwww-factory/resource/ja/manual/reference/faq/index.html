自動生成ツールの機能に関して

Entityクラスからテーブルを生成する機能は？

ありません。ERDツールなどでDB設計をし、DDL文を生成してデータベースにスキーマを作成してから Entity を自動生成するというやり方に特化しています。

O/Rマッパの機能に関して

ConditionBeanで one-to-many (1:n)の検索は？

Behavior の LoadReferrer 機能を利用します。load[子テーブルへの関連]() という形式のメソッドに Entity のリスト と 子テーブルに関する検索条件(CB) を指定することで、指定されたリスト(のそれぞれのEntity)が子テーブルのデータを保持するようになります。
