インストール手順

Eclipse - プラグインのインストール

更新サイトの登録

更新サイトに以下サイトを登録(名前は "Seasar Eclipse 3.3" など)します。

更新サイトのURL
http://eclipse.seasar.org/updates/3.3

インストール

登録したサイトからEMechaを選択してインストールします。 グループ化を解除せずに(デフォルト設定のまま)インストールすることを推奨します。
