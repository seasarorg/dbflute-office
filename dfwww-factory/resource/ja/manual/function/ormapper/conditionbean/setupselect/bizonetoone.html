SetupSelect(BizOneToOne)

${indexlist}

概要

基本概念

業務的one-to-one に対する SetupSelect(Relation) です。 固定条件に対してバインド変数がある場合は引数が定義されます。

現場フィット - 業務的one-to-one

会話上では、せっとあっぷせれくとびずわんとぅわん、業務的one-to-oneのせっとあっぷせれくと などと表現します。SetupSelect(Relation) という名前自体に、業務的one-to-oneの関連テーブルへの絞り込み条件も含まれますが、"業務的one-to-oneのテーブルに対して" ということを強調したい場合に明示的な表現として利用されます。

実装方法

setupSelect_[biz-one-to-one-table]() を呼び出します。 

e.g. SetupSelectの実装手順 (Eclipseでコード補完) {MEMBER_ADDRESS} @Java
Date targetDate = ...;
MemberCB cb = new MemberCB();
// 1. .seS まで打つと関連テーブル選択
// 2. MA (MemberAddress) で enter
cb.seSMA
--
cb.setupSelect_MemberAddressAsValid(targetDate);

メソッド仕様

基本的な仕様は通常の SetupSelect と同じです。バインド変数や別リレーションのカラムを利用した場合に特有の仕様が存在します。
